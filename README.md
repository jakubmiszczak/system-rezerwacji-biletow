# system-rezerwacji-biletow

System rezerwacji biletów do kina implementujący strukturę projektowego diagramu klas UML oraz wybrany przypadek użycia – usunięcie rezerwacji przez administratora.

Pełna dokumentacja projektu znajduje się w pliku:
**Dokumentacja.pdf**

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- React + Vite

## Requirements
- Java 17
- Node.js 18+

## Run locally

### Backend
```bash
cd backend

# Windows
mvnw.cmd spring-boot:run

# macOS / Linux
./mvnw spring-boot:run
```

Backend: http://localhost:8080

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend: http://localhost:5173
